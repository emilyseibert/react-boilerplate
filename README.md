# react-boilerplate

Basic React Boilerplate

```
npm install
npm start
```

## What's included
* React
* Linting
* Hot Reloading (Development)
* Building (Production)
* React Redux (example implemented)
* React Router (example implemented)
* A11y (Development)
* Sass (example implemented)
